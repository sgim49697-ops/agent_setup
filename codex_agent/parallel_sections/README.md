# parallel_sections

`parallel_sections`는 섹션별 작성 병렬화를 실험하는 워크스페이스입니다. 속도와 확장성은 좋아질 수 있지만, merge 단계가 약하면 문체와 정보 밀도가 들쭉날쭉해질 수 있습니다.

## 사용법

```bash
cd /home/user/projects/agent_setup/codex_agent/parallel_sections
codex
cd app && npm install && npm run dev
```

## 이 하네스의 핵심

- common research
- common outline
- parallel section writers
- merge review
