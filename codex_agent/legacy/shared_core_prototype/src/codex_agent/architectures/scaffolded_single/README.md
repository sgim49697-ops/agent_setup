# scaffolded_single - 계획/구현/검증 강제형

`scaffolded_single`은 단일 에이전트 구조의 단순함을 유지하면서도 `plan -> implement -> verify`를 명시적으로 강제합니다. 첫 버전의 기본형으로 가장 적합한 구조입니다.

## 기대 특성

- 강점: 범용성, 계획 가시성, 검증 루프 안정성
- 약점: 완전 자유 루프보다 약간 느릴 수 있음

## 실험 포인트

- 대부분의 시나리오에서 평균 점수가 가장 균형 잡혀야 한다.
- 베스트 디폴트 후보로 삼기 위해 실패율과 분산을 함께 본다.
