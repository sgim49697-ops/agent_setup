# sequential_pipeline

`sequential_pipeline`은 역할이 분명한 순차 워크플로를 실험하는 워크스페이스입니다. 단계가 명확해서 추적과 디버깅이 쉽지만, 중간에 잘못된 구조를 잡으면 뒤 단계가 그대로 끌려갈 수 있습니다.

## 사용법

```bash
cd /home/user/projects/agent_setup/codex_agent/sequential_pipeline
codex
cd app && npm install && npm run dev
```

## 이 하네스의 핵심

- researcher
- outliner
- writer
- reviewer
