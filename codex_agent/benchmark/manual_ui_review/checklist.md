# checklist.md - UX Benchmark v2 수동 검증 항목

## First screen

- 서비스 목적이 5초 안에 이해되는가
- 시작 행동이 명확한가
- 첫 화면 정보량이 과도하지 않은가

## Journey

- 현재 단계가 명확한가
- 다음 행동이 명확한가
- 단계 이동이 자연스러운가

## IA

- product UI와 evidence UI가 섞이지 않았는가
- 긴 정보는 접힘/분리되어 있는가
- 필수 산출물이 전체 여정 안에서 이해 가능한가

## Export

- final 단계와 export가 자연스럽게 연결되는가
- export 전/후 피드백이 이해 가능한가

## Recovery

- 에러 후 복구가 쉬운가
- retry/back 흐름이 설명되는가

## Differentiation

- 이 하네스만의 구조적 메타포가 느껴지는가
- 다른 하네스와 충분히 다른 interaction model을 갖는가
