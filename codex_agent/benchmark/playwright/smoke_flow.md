# smoke_flow.md - 공통 Playwright smoke flow

## 기본 흐름

1. 앱 진입
2. `Topic`, `Audience`, `Tone`, `Length` 입력 확인
3. `Generate post` 실행
4. `Research results`가 채워지는지 확인
5. `Outline`이 보이는지 확인
6. `Section drafts`가 보이는지 확인
7. `Review notes`가 보이는지 확인
8. `Final post`가 보이는지 확인
9. `Copy markdown` 액션이 존재하는지 확인

## 캡처 권장 상태

- initial
- loading
- populated
- review complete
- export ready
