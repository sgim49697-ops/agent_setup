# orchestrator_worker

`orchestrator_worker`는 동적 작업 분배와 통합 품질을 실험하는 워크스페이스입니다. 구조가 복잡해지는 대신, 화면/상태/콘텐츠를 역할별로 나눠 진척을 내기 좋습니다.

## 사용법

```bash
cd /home/user/projects/agent_setup/codex_agent/orchestrator_worker
codex
cd app && npm install && npm run dev
```

## 이 하네스의 핵심

- task decomposition
- worker ownership
- final integration
