# Tech Blog Generator Benchmark Starter

이 앱은 `codex_agent` 실험실의 공통 starter app입니다. 최종 제품은 `기술 블로그 포스트 자동 생성기`이며, 각 하네스는 이 동일한 출발점에서 Codex를 사용해 다른 오케스트레이션 방식으로 구현을 진행합니다.

## 개발 명령

```bash
npm install
npm run dev
npm run build
```

## 현재 상태

- 필수 입력 필드와 단계 계약을 설명하는 starter UI만 포함
- 실제 생성 파이프라인은 아직 구현되지 않음
- 최종 목표와 완료 기준은 상위 워크스페이스의 `spec/`와 `harness/` 문서를 참고
